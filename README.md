# ob-screenshot

A lightweight application for quickly taking screenshots in OpenBox

## 🚀 Features
- Easy screenshot capture with selection mode
- Saves screenshots automatically to `~/Screenshots` with timestamps.
- Suckless ui

## 📦 Requirements
- Python 3
- Scrot
- Tkinter
- Qt

## 🛠️ Installation
```bash
git clone https://github.com/w3techh/ob-screenshot.git

```bash
cd ob-screenshot
