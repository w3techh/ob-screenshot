# ob-screenshot

A lightweight application for quickly taking screenshots in OpenBox

## 🚀 Features
- Easy screenshot capture with selection mode
- Saves screenshots automatically to `~/Screenshots` with timestamps.
- Suckless ui

## 📦 Requirements
- Python 3
- Scrot
- Tkinter

## 🛠️ Installation
```bash
git clone https://github.com/w3techh/ob-screenshot.git
cd ob-screenshot

## 🖱️ Usage
```bash
python3 obs.py

📸 Take Screenshot – Click the button to select an area of your screen.
🪟 Minimize Window – The main window will minimize automatically while you select the area.
💾 Save Screenshot – Screenshots are saved in a Screenshots folder in your Home directory.
❌ Quit – Click Quit to close the application.
